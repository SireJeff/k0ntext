# Security Policy

## Supported Versions

| Version | Supported          | Notes                  |
|---------|--------------------|------------------------|
| 3.0.0+  | ✅ Yes             | Current stable version |
| < 3.0.0 | ❌ No longer supported |                       |

## Reporting a Vulnerability

If you discover a security vulnerability in K0ntext, please report it confidentially by emailing `security@sirejeff.com`.

## Disclosure Process

1. **Report**: Send vulnerability details to security@sirejeff.com
2. **Acknowledgment**: We'll acknowledge receipt within 24 hours
3. **Investigation**: We'll investigate the report within 48 hours
4. **Fix**: We'll work to resolve the issue promptly
5. **Disclosure**: We'll coordinate public disclosure

## Responsible Disclosure

We appreciate your help in keeping K0ntext secure. By following this policy, you help ensure that vulnerabilities are addressed before they can be exploited.

## Contact

For security-related questions or concerns, please contact us at security@sirejeff.com.
